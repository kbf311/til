# GitHub CLI ユーティリティスクリプト

このディレクトリには、GitHub CLI（`gh`）のセットアップや設定を自動化・簡易化するための Windows 用バッチファイル（`.bat`）が格納されています。

## スクリプト一覧

| ファイル名 | 概要 |
| :--- | :--- |
| [disable-issues.bat](./disable-issues.bat) | GitHub CLI を使用して、現在の GitHub リポジトリの Issues 機能を無効化します。 |
| [install.bat.bat](./install.bat.bat) | `winget` を使用して GitHub CLI をシステムにインストールします。 |
| [login.bat.bat](./login.bat.bat) | GitHub CLI で GitHub アカウントへログイン（認証）します。 |
